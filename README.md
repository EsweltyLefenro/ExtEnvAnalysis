# ExtEnvAnalysis

**ExtEnvAnalysis** — настольное приложение для пошагового анализа внешней среды и построения маркетинговой стратегии IT-предприятия.

Проект реализован как программное средство **«Анализ внешней среды»** в рамках выпускной квалификационной работы на тему:

> «Разработка программного обеспечения для симуляции построения маркетинговой стратегии IT-предприятия»

Главная идея приложения — провести студента по всей логике учебного кейса: выбрать сегмент IT-индустрии, заполнить PESTEL-анализ, определить факторы конкурентоспособности, сравнить собственную компанию с конкурентами, построить стратегические карты и получить итоговый PDF-отчет.

Для меня это, по сути, второй серьезный программный проект. Он уже заметно глубже раннего прототипа **EnvAnalysis**, но все еще отражает этап обучения и активного роста. Поэтому архитектура, организация кода и визуальная часть могут быть не самыми удачными или не полностью «промышленными». Главная ценность проекта — в том, что он доводит учебную методику до рабочего приложения.

## Назначение

Приложение предназначено для сопровождения учебной работы по построению маркетинговой стратегии IT-предприятия.

При ручном выполнении такого задания легко нарушить последовательность этапов, ошибиться в расчетах, потерять связь между таблицами, картами и выводами, а также получить отчеты разного формата. **ExtEnvAnalysis** решает эту задачу через пошаговый интерфейс, встроенные проверки, автоматический расчет рейтингов и генерацию PDF-отчета единой структуры.

Проект не является универсальной корпоративной системой стратегического планирования. Это локальное учебное приложение под один пользовательский сценарий и одну рабочую сессию.

## Возможности

- ввод профиля пользователя: ФИО, группа, уровень работы;
- выбор сегмента IT-индустрии из готового набора;
- проведение PESTEL-анализа по шести группам факторов;
- автоматическая подстановка факторов конкурентоспособности для выбранного сегмента;
- настройка весов факторов с контролем суммы `1.00`;
- оценка собственной компании и трех конкурентов по шкале от 1 до 10;
- ввод долей рынка с проверкой суммы не более 100%;
- расчет взвешенных рейтингов компаний;
- построение стратегических карт для пар активных факторов;
- ввод направлений развития и итоговых выводов;
- генерация, открытие и сохранение PDF-отчета;
- режим «Разработчик» с демонстрационным заполнением данных.

## Сценарий работы

Интерфейс построен как последовательность вкладок:

1. **Сегмент** — выбор направления IT-индустрии.
2. **PESTEL** — описание факторов внешней среды.
3. **Факторы и веса** — выбор факторов конкурентоспособности.
4. **Оценка компаний** — сравнение своей компании и конкурентов.
5. **Карты сравнения** — визуальный анализ стратегических карт.
6. **Итоги** — формулирование выводов.
7. **Отчет** — генерация PDF-документа.

Доступность следующих вкладок зависит от корректности предыдущих этапов. Это помогает пользователю не пропускать обязательные шаги и не формировать отчет с неполными данными.

## Методическая основа

В основе приложения лежат три элемента:

- **PESTEL-анализ** — структурирование внешней среды по политическим, экономическим, социальным, технологическим, экологическим и правовым факторам;
- **взвешенная оценка факторов** — расчет конкурентной позиции компаний с учетом значимости критериев;
- **стратегические карты** — визуальное сравнение компаний по парам факторов с учетом долей рынка.

Такой набор позволяет перейти от описания внешней среды к оценке конкурентной позиции и формулированию направлений развития.

## Режимы работы

- **Бакалавр** — основной режим выполнения учебного кейса.
- **Магистрант** — зарезервирован для дальнейшего развития и сейчас отключен.
- **Разработчик** — демонстрационный режим с автозаполнением данных.

Режим «Разработчик» нужен для быстрой проверки приложения и показа полного примера без ручного заполнения всех вкладок.

## Сборка и запуск

### Через Visual Studio

1. Установить **Visual Studio 2022** с workload для .NET desktop-разработки.
2. Открыть `ExtEnvAnalysis.sln`.
3. Дождаться восстановления NuGet-зависимостей.
4. Запустить проект `ExtEnvAnalysis`.

### Через .NET CLI

```powershell
git clone https://github.com/EsweltyLefenro/ExtEnvAnalysis.git
cd ExtEnvAnalysis
dotnet restore ExtEnvAnalysis.sln
dotnet build ExtEnvAnalysis.sln
dotnet run --project ExtEnvAnalysis/ExtEnvAnalysis.csproj
```

Приложение является WPF-программой, поэтому для запуска требуется Windows.

## Проверка сборки

Проект проверялся командой:

```powershell
dotnet build ExtEnvAnalysis.sln -p:UseAppHost=false
```

Сборка завершилась успешно, без предупреждений и ошибок.

## Статус проекта

Проект является рабочим учебным приложением и реализует основной сценарий ВКР: ввод исходных данных, PESTEL-анализ, настройку факторов и весов, оценку компаний, построение стратегических карт, формирование выводов и генерацию PDF-отчета.

При этом приложение не хранит промежуточное состояние между запусками, не поддерживает многопользовательскую работу и не подключается к внешним базам данных или сервисам. Это осознанное ограничение: программа создавалась под учебную задачу, где важны простота, локальность и соответствие фиксированной методике.

В дальнейшем проект можно развивать за счет сохранения состояния кейса, расширения режима преподавателя, более гибкой настройки факторов и выбора стратегических карт, включаемых в отчет.

## Связанный проект

Ранее идея анализа внешней среды была реализована в проекте **EnvAnalysis**. **ExtEnvAnalysis** можно рассматривать как более позднюю и переработанную версию этой идеи: с WPF-интерфейсом, пошаговым сценарием, стратегическими картами, валидацией данных и генерацией PDF-отчета.
